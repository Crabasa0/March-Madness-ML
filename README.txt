In our project we created multiple Jupiter notebooks to handle different tasks.
Madness1: Contains some initial data exploration, as well as our attempts to match team names from the Keras dataset to team names on sports-reference.
Project_Data: Contains most of our data formatting code, as well as some modelling
Vector-Subtraction: Contains formatting for vector subtraction strategy, as well as some more modelling. 
Tournament-Sim: Code to simulate a tournament using maximum likelihood estimation.
